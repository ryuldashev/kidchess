# Validation Plan

## Current State (v1.0)

**MVP Live.** Продукт работает, собираем usage data через Umami. Пока без активного маркетинга.

---

## Next Validation Phase (v1.2)

## TL;DR

Проверяем, готовы ли родители платить за AI-тренера по шахматам для малышей 3-8 лет. Запускаем лендинг с waitlist, гоним трафик из родительских комьюнити и таргета на родителей. Успех = 100+ signups за 2 недели при конверсии >3%. Если меньше — либо messaging не тот, либо рынок слишком узкий.

---

## Главная гипотеза

"Родители детей 3-8 лет готовы платить $3.99/месяц за приложение, которое научит ребёнка шахматам через AI-объяснения и адаптивные задачи."

## Критерий успеха

| Метрика | Fail | OK | Great |
|---------|------|-----|-------|
| Signups за 2 недели | <50 | 50-150 | >150 |
| Конверсия landing | <2% | 2-5% | >5% |
| Email open rate | <20% | 20-35% | >35% |
| "Would pay" в опросе | <30% | 30-50% | >50% |

## Validation тактика

1. **Где показываем:**
   - Telegram-чаты для родителей (мамские чаты, раннее развитие)
   - Facebook/Instagram таргет: родители 25-40, интересы: шахматы, раннее развитие, развивающие игры
   - Reddit: r/chess, r/parenting
   - Шахматные школы (холодный outreach)

2. **Кому показываем (ICP):**
   - Родители детей 3-8 лет
   - Интересуются развитием ребёнка
   - Уже пробовали учить шахматам или хотят начать
   - Готовы платить за образовательные приложения

3. **Что измеряем:**
   - Signups на waitlist
   - Ответы на follow-up опрос (5 вопросов)
   - Клики по "готов платить"
   - Качественный feedback в ответах

## Timeline

| Неделя | Действия |
|--------|----------|
| День 1-2 | Лендинг live, настроить аналитику |
| День 3-5 | Посты в 5-10 родительских чатах, первые 20-30 signups |
| День 6-7 | Запуск таргета ($50 бюджет), анализ первых данных |
| Неделя 2 | Итерация messaging, расширение каналов, follow-up опрос подписчикам |
| День 14 | Решение: go / pivot / kill |

## Pivot triggers

- Если signups < 30 и конверсия < 1% → проблема в messaging, итерировать landing
- Если signups 30-50 но "would pay" < 20% → продукт интересен, но не за деньги → pivot в freemium с рекламой?
- Если из шахматных комьюнити 0 интереса → возможно, не тот канал, попробовать EdTech комьюнити
- Если все signups только 5-8 лет, 3-4 года игнорят → сузить возраст до 5-8

## Опрос для подписчиков (Day 7)

1. Сколько лет вашему ребёнку?
2. Умеет ли ребёнок играть в шахматы сейчас?
3. Пробовали ли учить раньше? Что не получилось?
4. Сколько готовы платить за приложение? (0 / до $3 / до $5 / до $10)
5. Что самое важное в таком приложении?
